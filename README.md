# Image to ASCII Art

A very tiny python script that convert images to ASCII Art.

## Getting Started

You can use system-provided python 2 or install python 3 to run this script. 

### Prerequisites

Install Homebrew

```
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

Install Python 3 (If necessary)

```
$ brew install python3
```


### Running

Run through system-provided Python 2

```
python image-converter-ascii.py
```

Run through Python 3 (if installed)

```
python3 image-converter-ascii.py
```

Note: The image should be named 'test.jpg'.
